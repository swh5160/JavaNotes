##### 	7、运算符优先级

==优先级高的运算符会优先选择左右两边参与运算的表达式，并不是计算==

![image-20230207180444815](assets/image-20230207180444815.png)

```java
int a = 10；
boolean a = (a > 3 || ++a > 3 && a < 3)
//a=10 true;
```
